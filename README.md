# Offline-voice-command-switch
Using a VC-02 board we control a switch or any other electrical appliance, 
Required components<br/>
   VC-02 kit
   Relay(5V SPDT Relay)
   USB-B cable to connect the board to PC
   220 ohms resistor
   Diode (optional)
   Bulb 
   Bulb holder
   220 v Source
   NPN Transistor(BC547)
   Connecting wires
